שוק תחרותי: היצע, ביקוש והתערבות ממשלתית

סעיף זה מתמקד במודל היצע וביקוש בסיסי. כאן תלמדו כיצד לבנות פונקציות מצרפיות מתוך נתוני צרכנים ויצרנים בודדים. בנוסף, נחקור באופן ויזואלי כיצד התערבות ממשלתית בדמות מס קצוב, או פתיחת השוק לייבוא וקביעת מכסות, משפיעים על נקודת שיווי המשקל המקורית, על המחירים ועל הכמויות במשק.

חלק א': שיווי משקל בסיסי

ביקוש מצרפי (10 צרכנים):

Q_D = 10 * (60 - 0.2P) = 600 - 2P => P = 300 - 0.5Q

היצע מצרפי (20 יצרנים):

Q_S = 20 * (0.025P) = 0.5P => P = 2Q
חיתוך: 300 - 0.5Q = 2Q  ⇒  P* = 240, Q* = 120

חלק ב': מס של 60 ש"ח

הקשר החדש: P_d = P_s + 60

משוואה: 600 - 2(P_s + 60) = 0.5P_s

  • מחיר ליצרן (P_s): 192
  • מחיר לצרכן (P_d): 252
  • כמות חדשה (Q): 96
  • עודף יצרן חדש: 9,216

חלק ג': ייבוא ומכסות

מחיר עולמי 120 + הובלה 20 = 140.

במחיר P=140: Q_D = 600 - 2(140) = 320

עודף צרכן: 25,600

מכסת ייבוא לפי 22 יח' לצרכן:

סה"כ ביקוש: 220. מחיר תואם: P=190.

היצע מקומי ב-P=190: 95.

מכסה = 220 - 95 = 125 יח'

ביקוש היצע מקורי היצע עם מס מחיר ייבוא

תורת היצרן: עלויות ייצור ורווחיות

בחלק זה ננתח את החלטות הייצור של הפירמה התחרותית בהתבסס על פונקציית הייצור הנתונה שלה בטבלה. נתרגל חישוב של עלויות משתנות ממוצעות (AVC), עלויות כוללות ממוצעות (AC), ונראה כיצד הפירמה משתמשת בעלות השולית (MC) כדי להשוותה למחיר השוק ולמצוא את נקודת מקסימום הרווח שלה.

סעיף א': AVC ב-109 ק"ג

עלות משתנה ממוצעת = VC / Q

ב-109 ק"ג יש 5 עובדים.

VC = 5 * 150 = 750

AVC = 750 / 109 = 6.88

סעיף ב': AC ב-77 ק"ג

עלות כוללת ממוצעת = (VC + FC) / Q

ב-77 ק"ג יש 3 עובדים.

TC = (3*150) + 220 = 670

AC = 670 / 77 = 8.7

סעיף ג': ייצור במחיר 9 ש"ח

כלל: ייצר עד ש- MC = P. מחשבים MC.

MC מ-3 ל-4 עובדים: 150/20 = 7.5

MC מ-4 ל-5 עובדים: 150/12 = 12.5

עוצרים ב-4 עובדים (Q=97) כי 12.5 > 9.

רווח: (9*97) - ((4*150)+220) = 53

סעיף ד': מענק קבוע

המענק של 100 ש"ח מוריד את העלויות הקבועות (FC). כיוון שהוא אינו תלוי בכמות המיוצרת השולית (החל מהעובד ה-3), הוא אינו משנה את העלות השולית (MC). לכן, נקודת החיתוך MC=P נשארת זהה, ומחיר האיזון והכמות של הפירמה לא ישתנו.

מאקרו כלכלה: שיווי משקל במשק פתוח

שאלה זו בוחנת שיווי משקל מקרו-כלכלי במשק פתוח עם משטר שער חליפין נייד. המערכת דורשת פתרון של שתי משוואות מרכזיות במקביל: משוואת התוצר (מקורות ושימושים) ומשוואת מאזן התשלומים. לחצו על השלבים מטה כדי לעקוב אחר תהליך הפתרון של מערכת המשוואות.

נחשב תחילה את ההכנסה הפנויה ואז נציב ב-C:

Y_d = Y - T + TR = Y - 7000 + 300 = Y - 6700

כעת נסכום את כל הביקושים: AD = C + I + G + X - IM

AD = [4500 + 0.75(Y-6700)] + [1600 - 20000(0.06)] + 7000 + [400 + 300e] - [420 + 0.1Y - 200e]

AD = 6855 + 0.65Y + 500e

נשווה את התוצר לביקוש המצרפי שמצאנו בשלב 1:

Y = 6855 + 0.65Y + 500e
0.35Y = 6855 + 500e

קיבלנו משוואה עם שני נעלמים. נצטרך משוואה נוספת ממאזן התשלומים.

במשטר נייד: (X - IM) + CF = 0

[400 + 300e - 420 - 0.1Y + 200e] + [10000(0.06 - 0.02)] = 0
500e - 0.1Y - 20 + 400 = 0

נחלץ את 500e כדי להציב במשוואת התוצר:

500e = 0.1Y - 380

נציב את 500e בתוך משוואת התוצר משלב 2:

0.35Y = 6855 + (0.1Y - 380)
0.25Y = 6475
Y* = 25,900
e* = 4.42

חשבונאות לאומית: קשרי גומלין

נתרגל כאן מושגי יסוד בחשבונאות לאומית. נראה כיצד נתונים על חיסכון, מיסים והכנסה פנויה מאפשרים לנו לחלץ נעלמים מרכזיים כמו התוצר והצריכה הפרטית. לחצו על הנעלמים בטבלה כדי לגלות כיצד מסיקים אותם מהנתונים.

נתונים: מיסים (T=3100), חיסכון פרטי (Sp=1500), עודף יבוא (200), הכנסה פנויה (Yd=5200), חיסכון ממשלתי (Sg=200).

חישוב צריכה פרטית (C)
Sp = Yd - C
1500 = 5200 - 3700

החיסכון הפרטי הוא ההפרש בין מה שנשאר לאזרחים בכיס לבין מה שהם בחרו להוציא לצריכה.

חישוב תוצר נקי (Y)
Yd = Y - T + TR
5200 = 8300 - 3100

ההכנסה הפנויה היא התוצר (ההכנסה הלאומית) פחות המיסים שהממשלה גובה (בהנחת העברות TR=0).
