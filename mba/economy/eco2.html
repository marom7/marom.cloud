סימולטור שאלה 1: היצע וביקוש מצרפי

סעיף זה מהווה כ-23 נקודות מהמבחן והוא מבוסס לחלוטין על אלגברה פשוטה. המטרה כאן היא להבין את האלגוריתם: מעבר מפונקציית יחיד למצרפית, מציאת שיווי משקל (חיתוך), וחישוב שטחים (עודף יצרן/צרכן). עקוב אחר השלבים מטה, נסה לחשב בעצמך, ולחץ על הכפתורים כדי לראות את הפתרון ואת הביטוי הגרפי שלו.

נתוני הפתיחה

  • צרכנים: 10 צרכנים זהים. כל אחד: P = 300 - 5q
  • יצרנים: 20 יצרנים זהים. כל אחד: P = 40q

שלב 1: פונקציות מצרפיות (Aggregate)

כלל ברזל: תמיד מבודדים את הכמות (q) לפני שמכפילים במספר האנשים!

א. ביקוש מצרפי (Demand)

1. נבודד את q:
5q = 300 - P ⇒ q = 60 - 0.2P

2. נכפיל ב-10 צרכנים:
Q_D = 10 × (60 - 0.2P) = 600 - 2P

המשוואה הסופית: P = 300 - 0.5Q

ב. היצע מצרפי (Supply)

1. נבודד את q:
q = P / 40

2. נכפיל ב-20 יצרנים:
Q_S = 20 × (P / 40) = 0.5P

המשוואה הסופית: P = 2Q

שלב 2: שיווי משקל (Equilibrium)

משווים את הביקוש המצרפי להיצע המצרפי: Q_D = Q_S או P_D = P_S

חיתוך: 300 - 0.5Q = 2Q ⇒ 2.5Q = 300

כמות: Q* = 120
מחיר: P* = 240

שלב 3: התערבות ממשלתית - מס

הממשלה מטילה מס קצוב של 60 ₪ ליחידה. מס מקטין את ההיצע (מזיז את העקומה שמאלה/למעלה).

נוסחה: New Supply: P = (Old P) + Tax

היצע חדש: P = 2Q + 60

שיווי משקל חדש:

300 - 0.5Q = 2Q + 60 ⇒ 2.5Q = 240

  • כמות חדשה: Q = 96
  • מחיר לצרכן: Pd = 252
  • מחיר ליצרן: Ps = 192 (252 - 60)

התצוגה הגרפית (מודל השוק)

הגרף יתעדכן ככל שתתקדם בשלבים משמאל.

ביקוש מצרפי (D)
היצע מצרפי (S)
היצע + מס (S')

דף נוסחאות ממוקד ומבואר

ריכוז הנוסחאות הקריטיות שישמשו אותך במבחן. התמקד בהבנת המשתנים והיכן להציב אותם. זהו כלי העבודה המרכזי שלך לשאלות 2, 5 ו-6.

מיקרו כלכלה - תורת היצרן 💼
TC = VC + FC

עלות כוללת = עלות משתנה + עלות קבועה. (חשוב לשאלה 2).

AC = TC / Q

עלות ממוצעת. סך ההוצאות חלקי כמות התפוקה.

AVC = VC / Q

עלות משתנה ממוצעת. כלל ההשבתה: פירמה תייצר רק אם המחיר (P) גדול או שווה ל-AVC המינימלי.

Π = P × Q - TC

נוסחת הרווח (Π). פדיון כולל (מחיר כפול כמות) פחות סך העלויות.

מאקרו - מודל קיינסיאני 🏢
Y = C + I + G + X - IM

משוואת שיווי משקל (דו"ח מקורות ושימושים). זו המשוואה המרכזית לשאלה 5! מציבים את כל הפונקציות הנתונות ומוצאים את התוצר Y.

Yd = Y - T + TR

הכנסה פנויה = תוצר פחות מיסים פלוס העברות. מציבים את זה בתוך פונקציית הצריכה (C).

(X - IM) + CF = 0

מאזן תשלומים במשטר שער חליפין נייד. עודף היצוא (או גירעון היבוא) פלוס תנועות הון נטו (CF) חייבים להיות שווים לאפס. משתמשים בזה כדי למצוא את שער החליפין.

Sp = Yd - C | Sg = T - G

חיסכון פרטי (Sp) וחיסכון ציבורי/ממשלתי (Sg). (חשוב לשאלה 6).
