02
פרויקטים מוציאים זה את זה — NPV ו-IRR
17 נקודות

נתונים שני פרויקטים המוציאים זה את זה:

  • פרויקט A: השקעה (25,000) · תזרים שנה 1: 15,000 · שנה 2: 18,000
  • פרויקט B: השקעה (11,000) · תזרים שנה 1: 0 · שנה 2: 17,000

א. שרטטו את עקומות ה-NPV. ב. באילו שערי היוון יבוצע פרויקט A לפי כלל ה-NPV? ג. באילו שערים יבוצע B?

פתרון מפורט
שלב 1

מציאת IRR של פרויקט A — משוואה ריבועית

נציב X = 1/(1+IRR):

-25,000 + 15,000X + 18,000X² = 0 18X² + 15X - 25 = 0 X = [-15 ± √(225 + 1800)] / 36 = 0.833 IRR_A = 1/0.833 - 1 = 20%
שלב 2

מציאת IRR של פרויקט B

אין תזרים בשנה 1 → רק שנה 2:

-11,000 + 17,000/(1+IRR)² = 0 (1+IRR)² = 17,000/11,000 = 1.5454 IRR_B = 1.5454½ - 1 = 24.3%
שלב 3

נקודת חיתוך (r*)

משווים NPV של שני הפרויקטים:

-25,000 + 15,000/(1+r*) + 18,000/(1+r*)² = -11,000 + 17,000/(1+r*)² -14,000 + 15,000X + 1,000X² = 0 X = 0.8815 ⇒ r* = 13.44%
תשובה: לפי NPV: כש- r < 13.44% → בצע A · כש- 13.44% < r < 24.3% → בצע B · כש- r > 24.3% → דחה את שני הפרויקטים.
תובנה מרכזית

פרויקט A דורש השקעה גדולה יותר אבל מניב יותר בסה"כ. לכן הוא עדיף בריביות נמוכות. כשהריבית עולה, הזמן "יקר" יותר ופרויקט B (השקעה קטנה, החזר מהיר) הופך לעדיף. בנקודת r* שני הפרויקטים שווים.

משכנתאות 15 vs 30 שנה — תשלומים והצמדה
18 נקודות

רכישת דירת 4 חדרים במחיר 1,000,000 ש"ח. 200,000 ברשותכם, השאר במשכנתה. ריבית שנתית 10%, תשלומים שנתיים שווים (שפיצר), קרן וריבית.

א. גודל ההחזר השנתי במשכנתא ל-15 שנים? ל-30 שנים?
ב. כמה ישולם על חשבון הקרן בשנה 10 של המשכנתא ל-15 שנים?
ג. בכמה מופחתים סך התשלומים במשכנתא ל-15 שנים לעומת 30 שנים?
ד. בהנחה שהקרן והריבית צמודות לאינפלציה בשיעור 2% לשנה — מה ישולם על חשבון ריבית בשנה 3 של המשכנתא ל-30 שנים?

פתרון מפורט
א

תשלום שנתי לפי שפיצר

# משכנתא ל-15 שנים 800,000 = C · PVFA(10%, 15) C = 800,000 / 7.606 = 105,179 ש"ח # משכנתא ל-30 שנים 800,000 = C · PVFA(10%, 30) C = 800,000 / 9.427 = 84,863 ש"ח
ב

תשלום קרן בשנה 10 (משכנתא ל-15)

הטריק: יתרת הקרן בכל זמן = ערך נוכחי של התשלומים שנותרו.

# יתרת קרן בתחילת שנה 10 (לאחר תשלום שנה 9) Bal₉ = 105,179 · PVFA(10%, 6) = 458,082 # יתרת קרן בתחילת שנה 11 (לאחר תשלום שנה 10) Bal₁₀ = 105,179 · PVFA(10%, 5) = 398,711 # תשלום קרן בשנה 10 = הירידה ביתרה Principal₁₀ = 458,082 - 398,711 = 59,371 ש"ח
שיטה חלופית

חישוב הריבית: 10% · 458,082 = 45,808. תשלום קרן = 105,179 - 45,808 = 59,371. אותה תשובה!

ג

הפרש סך התשלומים

סך משכנתא 15 שנים = 15 · 105,179 = 1,577,685 סך משכנתא 30 שנים = 30 · 84,863 = 2,545,890 הפרש = 968,205 ש"ח חיסכון בריבית כשלוקחים ל-15
ד

ריבית בשנה 3 (משכנתא ל-30 עם הצמדה)

שלב א: מצא ריבית ריאלית — אם הריבית הנומינלית 10% והאינפלציה 2%:

r_R = 1.10/1.02 - 1 = 7.84%

שלב ב: עבדו עם תשלום ריאלי 84,863 (כמו לפני, רק שזה במונחים ריאליים), בנו לוח שפיצר וחשבו ריבית בשנה 3:

# בלוח השפיצר: יתרה אחרי שנה 2 → ריבית שנה 3 Bal₂_real = 84,863 · PVFA(7.84%, 28) ≈ 789,788 Interest₃_real = 0.0784 · 789,788 ≈ 61,920 # המרה לערכים נומינליים (אחרי 3 שנות הצמדה) Interest₃_nominal = 61,920 · (1.02)³ ≈ 65,712 ש"ח
סיכום:תשלום שנתי ל-15 שנים: 105,179 ש"ח · ל-30 שנים: 84,863 ש"ח · קרן בשנה 10: 59,371 · חיסכון: 968,205 · ריבית בשנה 3 עם הצמדה: ≈65,700.

תוכנית חיסכון "יתרון למתמיד" — ריביות משתנות
13 נקודות

זוג צעיר רוצה לרכוש מכונית בעוד 3 שנים. הם פותחים תוכנית חיסכון "יתרון למתמיד": הריבית בשנתיים הראשונות 6% נקובה לשנה (חישוב חודשי), ובשנה השלישית 12% נקובה (חישוב חודשי).

חשבו את הסכום שעל הזוג להפקיד מדי תחילת חודש (סה"כ 36 הפקדות), אם מחיר המכונית בעוד 3 שנים יהיה 90,000 ש"ח.

פתרון מפורט
שלב 1

חישוב ריביות חודשיות אפקטיביות

# שנתיים ראשונות: 6% נקובה ÷ 12 r_low = 6% / 12 = 0.5% חודשי # שנה שלישית: 12% נקובה ÷ 12 r_high = 12% / 12 = 1% חודשי
שלב 2

חישוב הערך העתידי של ההפקדות

24 הפקדות הראשונות (בריבית 0.5%) צריכות להמשיך לצבור גם בשנה השלישית בריבית 1%. 12 ההפקדות האחרונות (שנה 3) נצברות בריבית 1%.

# FVFA של תקופות החיסכון, כל הפקדה בתחילת חודש FVFA(0.5%, 24) = 25.432 FVFA(1%, 12) = 12.683 90,000 = C · (1.005) · FVFA(0.5%, 24) · (1.01)¹² + C · (1.01) · FVFA(1%, 12) 90,000 = C · 41.610
תשובה:הפקדה חודשית של C = 2,163 ש"ח.
למה (1.005) ולא 1?

ההפקדה הראשונה היא בתחילת החודש הראשון, ולכן צוברת ריבית גם באותו חודש. נוסחת FVFA רגילה מניחה הפקדה בסוף — לכן כופלים ב-(1+r) להמרה ל-"תחילת תקופה".

10 נקודות

חברת "מרחבים" הנפיקה ב-1/1/2018 אג"ח עם ערך נקוב 100 ש"ח, ריבית נקובה 8% לשנה, משולמת פעמיים בשנה (30/6 ו-31/12). פדיון: 31/12/2023.

קניתם את האג"ח ב-1/1/2019 כשהתשואה הנדרשת לפדיון הייתה 10.25% לשנה (אפקטיבי שנתי). מכרתם ב-1/1/2020 במחיר 87.58 ש"ח.

א. באיזה מחיר קניתם את האג"ח?
ב. מה שיעור התשואה שהשגתם בהנחה שהקופון ב-30/6/2019 הופקד בריבית 4.04% לשנה אפקטיבי?

א

חישוב מחיר הקנייה ב-1/1/2019

ב

תשואה על ההחזקה

קיבלנו 2 קופונים של 4 ש"ח (30/6/19 ו-31/12/19) + מחיר מכירה 87.58 ב-1/1/20.

תיק השקעות — תוחלת, סטיית תקן, ויחס תיק יעיל
24 נקודות

נתונות התפלגויות התשואה של שתי מניות:

  • מניה A: P=0.2 → 16%, P=0.4 → -8%, P=0.4 → 10%
  • מניה B: P=0.2 → 36%, P=0.4 → 16%, P=0.4 → -4%
  • מקדם מתאם ρ(A,B) = 0

א. תוחלת וסטיית תקן של כל מניה ושל תיק 60% A + 40% B.
ב. דרגו לפי קריטריון MV (תוחלת-שונות).
ג. מצאו הרכב תיק עם סטיית תקן 9%.

פתרון מפורט
א

תוחלת וסטיית תקן של כל מניה

E(R_A) = 0.2·16 + 0.4·(-8) + 0.4·10 = 4% Var(R_A) = 0.2(16-4)² + 0.4(-8-4)² + 0.4(10-4)² = 100.8 SD(R_A) = 10.04% E(R_B) = 0.2·36 + 0.4·16 + 0.4·(-4) = 12% Var(R_B) = 224 ⇒ SD(R_B) = 14.97% # תיק 60/40 E(R_P) = 0.6·4 + 0.4·12 = 7.2% Var(R_P) = 0.36·100.8 + 0.16·224 + 0 = 72.128 SD(R_P) = 8.49%
ב

דירוג לפי MV

נמקם את שלושת התיקים במרחב E(R) - SD(R):

  • A: (10.04%, 4%) · B: (14.97%, 12%) · P: (8.49%, 7.2%)
  • P עדיף על A: סטיית תקן נמוכה יותר וגם תוחלת גבוהה יותר → P ≻ A
  • בין P ל-B: B נותן יותר תשואה אבל גם יותר סיכון → לא ניתן להכריע
  • בין A ל-B: B נותן יותר תשואה אבל יותר סיכון → לא ניתן להכריע
ג

הרכב תיק עם סטיית תקן 9%

9² = W_A² · 100.8 + (1-W_A)² · 224 + 0 81 = 100.8W_A² + 224(1-2W_A+W_A²) 324.8W_A² - 448W_A + 143 = 0 # פתרון משוואה ריבועית W_A = [448 ± √(200,704 - 185,866)] / 649.6 W_A = 0.502 או 0.878
שני פתרונות אפשריים

0.502/0.498 (תיק יותר B): E(R) = 8% · 0.878/0.122 (תיק יותר A): E(R) = 5%. שני הרכבים נותנים אותה סטיית תקן 9%.

סיכום:שני הרכבי תיק אפשריים: 50.2% A + 49.8% B (תוחלת 8%) או 87.8% A + 12.2% B (תוחלת 5%).
