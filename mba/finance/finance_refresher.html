1 ערך הזמן של הכסף הבסיס

שקל היום > שקל מחר. ארבע נוסחאות — סכום בודד או סדרה, קדימה (FV) או אחורה (PV).

סכום חד-פעמי
FV = PV × (1 + r)^n   |   PV = Cn / (1 + r)^n
סדרת תשלומים קבועים
PV = C × [ 1 − 1/(1+r)^n ] / r  =  C × PVFA(r,n)
FV = C × [ (1+r)^n − 1 ] / r  =  C × FVFA(r,n)
סדרה אינסופית (פרפטואיטי):   PV = C / r
דוגמה
מפקידים 10,000 ₪ ל-8 שנים בריבית 7%. כמה יהיה?
FV = 10,000 × (1.07)^8 = 17,182 ₪
💡
הפקדה בתחילת תקופה? הנוסחה הרגילה מניחה סוף תקופה. אם זה בתחילת כל תקופה — כפול הכל ב-(1+r).
⚠️
ריבית משתנה? אסור PVFA/FVFA! הן רק לריבית קבועה. משרשרים כל תקופה בנפרד.

2 תקופות קצרות מהשנה

כשהחישוב חודשי/רבעוני — מתאימים את n ואת הריבית.

n = t × m   (שנים × תקופות בשנה)
ריבית לתקופה: rm = i / m
אפקטיבית שנתית: R = (1 + i/m)^m − 1
מעבר אפקטיבי→קצר: rm = (1 + R)^(1/m) − 1
דוגמה
ריבית נקובה 12% לשנה, חישוב חודשי. מה האפקטיבית השנתית?
rm = 12%/12 = 1%
R = (1.01)^12 − 1 = 12.68%
🎯
כלל הזהב: ריבית חודשית הולכת רק עם n בחודשים. ריבית שנתית רק עם n בשנים. תבדוק את זה לפני כל היוון.

3 ריבית ריאלית ונומינלית

נומינלי = כולל אינפלציה. ריאלי = מנוכה אינפלציה.

נוסחת פישר
1 + rN = (1 + rR) × (1 + h)
rR = (1 + rN) / (1 + h) − 1   (h = אינפלציה)
דוגמה
ריבית נומינלית 10%, אינפלציה 20%. מה הריאלית?
rR = 1.10 / 1.20 − 1 = −8.33%
💡
עקביות: תזרים צמוד → היוון בריבית ריאלית. תזרים לא צמוד → ריבית נומינלית. אסור לערבב.

4 הלוואות ולוחות סילוקין

שפיצר — תשלום קבוע
C = Loan / PVFA(r, n)

ריבית = יתרת קרן × r  ·  קרן = C − ריבית  ·  יתרה חדשה = יתרה − קרן

🔥
טריק יתרת הקרן (חוסך לוח שלם!):
Balance_t = C × PVFA(r, n−t)
כמה שולם ע"ח קרן בשנה 10? פשוט Balance₉ − Balance₁₀.
לוח רגיל — קרן קבועה
קרן לתקופה = Loan / n  ·  תשלום = קרן קבועה + (יתרה × r)
צמודות למדד

1) בונים לוח בערכים ריאליים (ריבית ריאלית). 2) מכפילים כל תשלום במדד הצבור:

תשלום נומינלי = C_real × (1 + h)^t
הלוואות גישור

בולט: רק ריבית שוטפת, קרן בסוף. בלון: הכל (קרן+ריבית) בסוף. גרייס: דחיית תשלומים, הריבית נצברת.

5 הערכת אגרות חוב

מחיר אג"ח = PV של כל הקופונים + PV של הערך הנקוב.

P = C × PVFA(r, n) + F / (1 + r)^n   |   C = i × F
יחססטטוסמחיר
r = iפאריP = F
r > iניכיוןP < F
r < iפרמיהP > F
⚠️
קופון חצי-שנתי: n = שנים×2  ·  C = (i/2)×F  ·  r חייב להיות אפקטיבי חצי-שנתי: r½ = (1+R)^(1/2) − 1. לא לחלק ב-2 סתם!
סוגים נוספים

אפס קופון: P = F/(1+r)^n.  מק"מ: P = 100/(1+R)^(days/365).

תשואת החזקה (HPR)
HPR = (P2 + C − P1) / P1

6 תקצוב הון — NPV / IRR / PI

NPV — ערך נוכחי נקי
NPV = −I0 + Σ CFt/(1+r)^t   (קבוע: −I0 + CF×PVFA)

כלל: NPV > 0 → כדאי. NPV < 0 → לא כדאי.

IRR — שיעור תשואה פנימי

ה-r שמאפס את ה-NPV. כלל: IRR > r → כדאי.

🔥
IRR כשיש 2 תקופות (משוואה ריבועית): הצב X = 1/(1+IRR), פתור −I0 + CF1·X + CF2·X² = 0 בנוסחת השורשים, ואז IRR = 1/X − 1.
PI — מדד רווחיות
PI = PV / I0 = 1 + NPV/I0   (PI > 1 → כדאי)
⚠️
פרויקטים שמוציאים זה את זה: אם IRR סותר את NPV — תמיד מחליטים לפי NPV הגבוה.

7 בניית תזרים מזומנים השאלה הגדולה · 20-32 נק'

בונים טבלה שורה-שורה, ואז עושים לזה NPV. אל תיבהל מהגודל.

3 עקרונות קשיחים
עיקרוןמה עושים
עלות שקועהכבר הוצא בעבר (ייעוץ, מחקר) → לא נכלל בכלל
עלות אלטרנטיביתנכס קיים שיכל להימכר → שוויו נחשב כהשקעה
תזרים תוספתירק שינויים מהפרויקט. קניבליזציה → מפחיתים
מבנה הטבלה (סדר השורות)
  הכנסות (כמות × מחיר)
− הוצאות תפעוליות
− ירידה בהכנסות קיימות
− פחת לצורכי מס  (D = עלות / שנות פחת)
= רווח לפני מס
− מס הכנסה
= רווח אחרי מס
+ החזר פחת ←← אל תשכח!
−/+ השקעה/החזר הון חוזר  (בסוף חוזר במלואו)
− השקעה בציוד (זמן 0)
+ תמורה ממכירת ציוד (שנה אחרונה)
−/+ מס על רווח/הפסד הון
= תזרים נקי (NCF) → ל-NPV
🔥
מס על רווח הון בסוף:
ע"ב = עלות − פחת שנצבר  ·  רווח הון = מחיר מכירה − ע"ב  ·  מס = רווח × 25%
⚠️
שים לב לשני שיעורי מס שונים! מס חברות שוטף (30%) מול מס רווחי הון (25%). אל תתבלבל.

8 תיקי השקעות (אי-ודאות)

תוחלת: E(R) = Σ Pi × Ri
שונות: Var = Σ Pi × (Ri − E(R))²
סטיית תקן: SD = √Var  (מדד הסיכון)
קריטריון תוחלת-שונות (MV)

נכס א' דומיננטי על ב' אם: תוחלת גבוהה/שווה וגם סטיית תקן נמוכה/שווה (לפחות אחד ממש לטובתו). שונא סיכון בוחר תמיד את הדומיננטי.

🎯 טיפים אחרונים ל-65 ומעלה

  • ניהול זמן: 30 דקות בדיוק לשאלה. נתקעת? תעבור הלאה ותחזור.
  • הצג דרך פתרון: נוסחה + הצבה + טבלה = 70-80% מהניקוד גם אם הטעית במחשבון. אל תכתוב רק שורה סופית.
  • עקביות יחידות: לפני כל היוון — "הריבית והזמן באותן יחידות?"
  • שאלת התזרים: צייר טבלה לפי המבנה, מלא שורה-שורה. תזכור: החזר פחת, החזר הון חוזר.
